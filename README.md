# hexo-markmap
Innsert a mindmap by markdown in your hexo blog by markmap.

在你的博客中使用markdown插入思维导图，使用markmap。

# Install
```
npm install hexo-markmap
```

# Usage
```
{% markmap %}
# Testa
## test1
## test2
# Testb
## test1
## test2
{%endmarkmap%}
```
